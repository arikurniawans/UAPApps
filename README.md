* SIAKAD (Sistem Informasi Akademik)
* SIMKEU (Sistem Informasi Manajemen Keuangan)
* SIMPEG (Sistem Informasi Kepegawaian)
* NELITI UAP (Sistem Informasi LPPM UAP)
* PELANGI (Aplikasi Pembelajaran diatas Langit)
